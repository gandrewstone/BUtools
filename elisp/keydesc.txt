PRINT ME OUT FOR QUICK REFERENCE

F1: load
F2: save
F3, alt-left arrow: prev file (alphabetically)
F4, alt-right arrow: next file (alphabetically)
F5: rename buffer
F6: switch to buffer
F7: compile C, run Python
F8: next error (if in compile buffer)
F9: debug or show debug window
F10: shell or show shell window  (for multiple shells: F10, rename with F5, then F10 again)

Ctrl-f1 to Ctrl-f6: switch to buffers 1 thru 6
Ctrl-f7: switch to the compilation buffer
Ctrl-f9: switch to the debug buffer
Ctrl-f9: switch to the shell buffer

Alt-g: goto line
Alt-\: split window vertically
Alt--: split window horizontally
Alt-0: remove split window

Ctrl-f: search
Ctrl-s: search
Ctrl-r: search backwards
Ctrl-z: undo
Ctrl-v: paste
Ctrl-c: copy

Ctrl-i indent region
Ctrl-tab: clang format region

Ctrl-q: CSCOPE integration

Ctrl-q q: recalculate cscope tags
Ctrl-q s: cscope find symbol
Ctrl-q d: cscope find definition
Ctrl-q f: cscope find file
Ctrl-q i: cscope find include
Ctrl-q c: cscope find caller
Ctrl-q t: cscope find text
Ctrl-q w: switch to a new target window

In cscope buffer:
<CR> or mouse click -- goes to the referenced file/line
<SPACE> views the referenced file/line
alt-< previous query page
alt-> next query page

Ctrl-e: git integration
Ctrl-e s: git status (most commands only work from this screen)
Ctrl-e b: git blame toggle (works in a file context)
